You can find the webhook URL in your Solano CI Dashboard (see Solano CI Docs: Webhook URL for more information):
  1. Click on the gear icon next to the repository you wish to configure.
  2. Select "CI Setup" from the left side menu.
  3. Depending on how your repository is configured:
    • GitHub:Use https://hooks.solanolabs.com/1/build/<GITHUB SERVICE HOOK TOKEN>
    • Others:The webhook URL is displayed under "POST-COMMIT WEBHOOK URL". Note that the full URL contains an embedded authentication token as the last component of the URL path.
